# Phase 4 — SCALE (J+42 et au-delà)

**Objectif :** monter en volume sur ce qui a prouvé ROI positif.

## Lookalikes & scaling pub

- [ ] **Lookalike 1%** sur les meilleurs convertis (pixel + email engagés)
  - Source minimum 100-300 convertis (sinon LAL inefficace)
- [ ] Scale budget +30%/semaine sur les campagnes positives
- [ ] **Pause** systématique des créatives sous benchmark (CTR, CPL)
- [ ] Renouvellement créatif toutes les 2 semaines (anti-fatigue)
- [ ] Élargir progressivement à LAL 3%, 5%, 10% selon les performances

## Email — cadence régulière

- [ ] Newsletter hebdo (jour fixe, ex: jeudi 10h pour B2B FR)
- [ ] Séquences automation activées :
  - Welcome (J+0)
  - Anniversaire / date fixe
  - Abandon panier ou abandon formulaire
  - Réengagement à J+30/60/90
- [ ] Segmentation comportementale :
  - Engagés (open + clic dans les 30j)
  - Dormants (pas d'open dans 60j)
  - Perdus (pas d'open dans 180j)
- [ ] Réactivation des dormants à J+90 (campagne dédiée, sinon nettoyer)

## SEO — itération

- [ ] Analyse performances Search Console (clic / impression / position)
- [ ] Optimisation contenus moyens (rewriting, ajout internal links)
- [ ] Nouveaux clusters de mots-clés selon ce qui ranke
- [ ] Surveillance des chutes de position (backlinks toxiques, pénalités)
- [ ] Backlinks qualité (relations presse, guest posts, partenariats)

## Influence — activation

- [ ] Lancement collaborations avec influenceurs sélectionnés
- [ ] Code promo / lien affilié trackés (Switchy)
- [ ] Reporting et reconduction des plus performants
- [ ] Diversifier : nano (1k-10k), micro (10k-100k), macro (100k+) selon budget

## Bot scaling

- [ ] Augmentation graduelle volume bots (max +20%/semaine)
- [ ] Multi-comptes pour répartir le risque (3-5 comptes par plateforme)
- [ ] Diversification fingerprints (multi-IP, multi-browsers)
- [ ] Rotation des messages pour éviter détection patterns

## Phoning — passage à l'échelle

- [ ] Externaliser à un call center spécialisé si volume >50 leads/jour
- [ ] Scripts AB-testés
- [ ] Rapport quotidien : appels passés, contactés, qualifiés, RDV pris

## Optimisation continue (cadence)

- [ ] **Hebdo** : revue performances toutes campagnes actives
- [ ] **Bi-mensuel** : audit créatif (renouvellement)
- [ ] **Mensuel** : analyse ROI par canal + ré-allocation budget
- [ ] **Trimestriel** : audit SEO complet + audit délivrabilité

## Multi-sociétés — quand passer à la suivante

À J+56 (= 8 semaines), si la société actuelle est en Phase 4 stable :
- ROI positif identifié sur au moins 2 canaux
- Pipeline de leads autonome (pas besoin d'intervention quotidienne)
- Équipe contenu rodée

→ Démarrer une nouvelle société (Phase 1) en parallèle. Cadence cible : nouvelle société tous les 1-1.5 mois.
