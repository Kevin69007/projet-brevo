# Phase 2 — MATURATION + PRODUCTION (J+7 à J+28)

**Objectif :** pendant que les chronos tournent (warmup, pixels, SEO indexing), produire **massivement** les contenus et assets pour les phases d'activation.

## Email — passif (laisser tourner)

- [ ] Check Allegrow hebdo (J+14, J+21)
- [ ] Vérifier rapports DMARC dans `dmarc@<societe>.com` (signal que les receveurs voient bien le domaine)
- [ ] **NE PAS envoyer de campagne**. Patience absolue.

## Email — préparation campagnes (active)

- [ ] Rédiger **5-10 templates email** :
  - Welcome (J+0 inscription)
  - Contenu valeur (éducatif, nurturing)
  - Offre (call-to-action principal)
  - Relance (rappel non-engagés)
  - Désinscription propre (RGPD)
- [ ] Créer les **séquences automation** dans Brevo (sans les activer encore)
- [ ] Préparer les **listes de segmentation** dans Brevo (par vertical, par fraîcheur, par engagement)
- [ ] Définir les **règles de désinscription** + lien clair (RGPD)

## Social — production contenu

- [ ] Créer **30 jours de posts** organiques (Meta + LinkedIn + TikTok minimum)
- [ ] Programmer dans Metricool
- [ ] Démarrer publication dès J+7 pour faire chauffer les comptes
- [ ] Activer bots de commentaires/likes ciblés avec **prudence** :
  - 20-30 actions/jour les 14 premiers jours
  - Pas plus de 50/jour ensuite

## Pub — créatives à préparer

- [ ] **Brief créatif** par persona / par vertical
- [ ] Production **5-10 visuels** par audience (statique + carrousel + vidéo court)
- [ ] **Copies** de 3 longueurs (court / moyen / long) pour A/B test
- [ ] Préparer audiences custom dans Meta/Google (sans encore lancer)
- [ ] Hooks vidéo de 3 sec, 5 variantes par campagne

## Scraping — exécution & nettoyage

- [ ] Lancer scraping sur le périmètre cible
- [ ] Validation des emails (Email List Validation API)
- [ ] Dédoublonnage croisé avec BDD existante
- [ ] Enrichissement (Dropcontact, Hunter, FullContact selon vertical)
- [ ] Import segmenté dans Brevo (sans envoyer encore)

## SEO — production

- [ ] Publication articles selon le plan éditorial (rythme régulier signal Google)
- [ ] Maillage interne entre articles
- [ ] Soumission sitemap à Google Search Console
- [ ] Première campagne backlinks (annuaires qualité + guest posting)

## Influence & partenariats

- [ ] Identifier 20-30 micro/nano-influenceurs du vertical (Modash, Heepsy, recherche manuelle)
- [ ] Outreach personnalisé (cycle de négo 2-6 semaines)
- [ ] Préparer briefs et conditions

## SMS / WhatsApp / Postal — préparation

- [ ] Configurer **Brevo SMS** (volumes faibles) ou solution dédiée (Sarbacane, Esendex)
- [ ] **WhatsApp Business API** (Brevo, Twilio, WATI) — **délai validation Meta 5-10 jours**
- [ ] **Postal** : choisir prestataire (Mediapost, La Poste Pro), template visuel, base d'adresses
- [ ] Scripts respectant la réglementation (opt-in, opt-out, mentions légales)

## Validation Phase 2 — Critères pour passer Phase 3

- ✅ Warmup spam rate <5% sur 7 jours consécutifs (gate)
- ✅ 5-10 templates email prêts
- ✅ 30 jours de posts sociaux programmés
- ✅ 5-10 créatives pub par audience prêtes
- ✅ BDD scrapée + validée + segmentée dans Brevo
- ✅ 5-10 articles SEO publiés
- ✅ WhatsApp Business validé Meta
- ✅ Influenceurs identifiés (négos en cours)

Sans ces éléments, l'activation Phase 3 sera bâclée et tu perdras du budget.
