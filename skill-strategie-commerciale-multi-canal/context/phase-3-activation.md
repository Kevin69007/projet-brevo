# Phase 3 — ACTIVATION CONTRÔLÉE (J+28 à J+42)

**Objectif :** premiers envois réels, à VOLUME LIMITÉ, pour valider et collecter de la data sans cramer la reputation.

## Email (post-gate warmup)

- [ ] **Gate validé** : 7 jours consécutifs spam rate <5% sur Allegrow
- [ ] Première campagne sur **petite portion qualifiée** (200-500 contacts max)
- [ ] Monitoring fin :
  - Open rate (cible >25% B2B)
  - Click-through (cible >2%)
  - Bounce rate (cible <2%)
  - Plainte (cible <0,1%)
  - Désinscription (cible <0,5%)
- [ ] Si OK → 2ème campagne sur 1 000-2 000 contacts
- [ ] Si OK → scaling progressif (max +50% de volume par envoi)

## Pub Meta / Google / TikTok / LinkedIn — lancement

- [ ] **Pub Bdd custom audience** : upload BDD scrapée vers Meta/LinkedIn → conversion ratio attendu 30-60% match
- [ ] **Pub froide ciblage intérêt** sur petite audience (test 50-100€/jour par campagne pour collecter signaux)
- [ ] **Retargeting visiteurs site** (pixel mûr depuis J+14, suffisamment de data depuis J+28)
- [ ] Test 3-5 créatives en CBO (Campaign Budget Optimization)
- [ ] Frequency cap : 2-3 impressions/personne/semaine en froid

## SMS / WhatsApp première vague

- [ ] Envoi SMS sur **opt-in confirmés uniquement** (pas de scraping pour SMS — risque légal majeur)
- [ ] WhatsApp templates validés Meta (HSM)
- [ ] Tracking via Switchy
- [ ] Horaires respectueux : 10h-12h ou 14h-19h, jamais soir/week-end pour B2B

## DM réseaux sociaux

- [ ] **Bots de scraping LinkedIn** : extraction prospects + visites profil + connexion
- [ ] **Bots d'activité** : like + commentaire + invitation (volume limité 20-30/jour)
- [ ] Cold outreach LinkedIn manuel sur top-prospects (max 10/jour pour éviter détection)
- [ ] Instagram DM via outil officiel ou semi-manuel

## Phoning

- [ ] Sur les leads qui ont **engagé** (open email + clic ou inscription)
- [ ] Script qualifié, pas de cold call brut (sauf si BDD ultra-qualifiée + contexte légal OK)
- [ ] Conformité Bloctel obligatoire pour B2C en France

## Postal

- [ ] **Sur top 5-10% de la BDD** uniquement (coût élevé, 0,80-2€ par envoi)
- [ ] Cibler les leads chauds non répondus aux autres canaux
- [ ] Tracking : QR code, code promo unique, URL personnalisée

## Validation Phase 3 — Critères pour passer Phase 4

- ✅ Email : 3-5 campagnes envoyées sans dégradation reputation (spam rate stable <5%)
- ✅ Pub : 50+ conversions trackées (seuil minimum pour lookalike)
- ✅ SMS/WA : taux de plainte <0,5%
- ✅ Phoning : taux de conversion lead engagé → RDV connu (baseline pour scale)
- ✅ Postal : ROI calculé sur la première vague (souvent visible à J+14 après envoi)

Sans ces données, le scaling Phase 4 = pari aveugle.
